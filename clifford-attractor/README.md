# Clifford Attractor

## Quickstart

If necessary, install [Anaconda](https://docs.anaconda.com/anaconda/install/) or [Miniconda](https://docs.conda.io/en/latest/miniconda.html) first, please.

- `make init`.
- `make rstudio`.
